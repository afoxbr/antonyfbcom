About the site

This site is built using Python in Django. It's deployed on an Alpine Linux virtual machine using Gunicorn and Nginx. It's monolithic, basic and sturdy in design, much like the author.

    Planned features

  • Blog posts searchable by tag
  • Blog posts searchable by text/title content

  • A git repo to self-host my projects' code

About the author

Antony Fox-Bramwell is a 30-something husband and father from the East of England. He started coding shortly after his 31st birthday, but began writing about himself in the 3rd person long before that. He's been an enthusiastic fan of open source software since his late teens, and regrets not getting into tech when he was younger.
